# payloads-ps4-900

PS4 Payloads Collection FW9.00<br>


<br>

## Setup on PI
```sh
sudo rm -f -r ~/payloads
sudo rm -f -r /boot/firmware/PPPwn/payloads
git clone https://github.com/Systemx23x/payloads-ps4-900 ~/payloads
cd ~/
sudo cp -r payloads /boot/firmware/PPPwn/
```
